## 3. Gestión de Clientes

#### Paso 1: Ir a la sección "Clientes"

- **Ver** la interfaz de gestión de clientes

- **Hacer clic** en "➕ Agregar Cliente"

## 4. Registro de Préstamos

### 💰 Nuevo Préstamo

#### Paso 1: Acceder al formulario
- **Hacer clic** en "💰 Préstamos" en el menú
- **Hacer clic** en "➕ Registrar Préstamo"

#### Paso 2: Seleccionar cliente
- **Elegir** cliente de la lista desplegable
- **O buscar** cliente por nombre

#### Paso 3: Ingresar datos del préstamo
```
💵 Monto Base: Ej: 1.500.000
📅 Fecha: Fecha del préstamo
📝 Descripción: Motivo del préstamo (opcional)
```

#### Paso 4: Observar cálculos automáticos
El sistema calculará automáticamente:
- ✅ **20% Adicional:** 20% del monto base
- ✅ **Total con 20%:** Monto base + 20%
- ✅ **Unidad del monto:** Clasificación automática

#### Paso 5: Guardar préstamo
- **Hacer clic** en "💾 Registrar Préstamo"
- **Confirmar** registro exitoso

### 📊 Visualización en Tiempo Real

#### Indicadores automáticos:
- **Monto Base:** Formato con puntos (ej: 1.500.000)
- **20% Adicional:** Cálculo automático
- **Total Final:** Suma completa
- **Unidad:** "Pesos" para montos ≥ 1.000

#### Validaciones automáticas:
- ✅ **Formato numérico:** Separación de miles
- ✅ **Cálculos:** Precisión matemática
- ✅ **Campos obligatorios:** Validación en tiempo real

## 5. Sistema de Cálculos

### 🧮 Cálculo Automático del 20%

#### ¿Cómo funciona?
1. **Ingresar** monto base (ej: 1.000.000)
2. **Ver** cálculo automático:
   - **20% Adicional:** 200.000
   - **Total Final:** 1.200.000

#### Formatos soportados:
- ✅ **Números pequeños:** 500 → 100 → 600
- ✅ **Números medianos:** 15.000 → 3.000 → 18.000
- ✅ **Números grandes:** 1.500.000 → 300.000 → 1.800.000

### 💵 Formato de Moneda Colombiana

#### Características:
- **Separador de miles:** Punto (.)
- **Sin decimales:** Solo números enteros
- **Ejemplos:**
  - 1.000 = Un mil pesos
  - 15.000 = Quince mil pesos
  - 1.500.000 = Un millón quinientos mil pesos

#### Clasificación por unidades:
- **1.000+:** "Pesos"
- **100-999:** "Centenas"
- **10-99:** "Decenas"
- **0-9:** "Unidades"

## 10. Solución de Problemas

### ❌ Problemas Comunes y Soluciones

#### 1. No puedo iniciar sesión
**Posibles causas:**
- ✅ **Email/contraseña incorrectos**
- ✅ **Usuario no registrado**
- ✅ **Sesión activa previa**

**Soluciones:**
- 🔍 Verificar credenciales
- 📝 Registrar nueva cuenta
- 🗑️ Limpiar datos del navegador

#### 2. Cálculos no aparecen
**Posibles causas:**
- ✅ **Campos vacíos**
- ✅ **Formato incorrecto**
- ✅ **JavaScript desactivado**

**Soluciones:**
- 🔢 Ingresar monto válido
- 🧹 Limpiar campo y reintentar
- 🔄 Recargar página

#### 3. WhatsApp no se abre
**Posibles causas:**
- ✅ **Número no registrado**
- ✅ **Formato incorrecto**
- ✅ **WhatsApp no instalado**

**Soluciones:**
- 📱 Verificar número en cliente
- ✅ Actualizar datos del cliente
- 📲 Instalar WhatsApp

#### 4. Datos no se guardan
**Posibles causas:**
- ✅ **Conexión inestable**
- ✅ **Navegador no compatible**
- ✅ **LocalStorage lleno**

**Soluciones:**
- 🌐 Verificar conexión
- 🔄 Usar navegador compatible
- 🗑️ Limpiar datos del navegador

- **📱 Móvil:** Compatible con navegadores móviles
